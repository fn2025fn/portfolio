自己紹介

前職では医療事務として勤務していましたが、産休中にコーディングに出会い、
「自分の書いたコードが画面に表現される楽しさ」と「それが多くの人に情報として届くこと」に魅力を感じました。
子育てが落ち着いたタイミングで本格的に学びたい気持ちが高まり、TechCampを受講。
学習を進める中で、エラーを乗り越えた時の達成感や、機能が完成するたびの喜びが大きなモチベーションになっています。
今後は、ユーザーにとって使いやすく、伝わるものをつくれるエンジニアとしてキャリアを築いていきたいと考えています。

制作実績

オリジナルアプリ：お手伝い管理アプリ

Ruby on Rails / PostgreSQL / Devise / JavaScript
親が子どものお手伝い状況を管理できる「親子向けお手伝い記録・思い出共有アプリ」です。お手伝いの記録に応じてごほうびポイントが加算され、写真付きで思い出も残せます。

アプリを見る

クライアント案件：Webページコーディング
